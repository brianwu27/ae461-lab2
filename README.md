fringe_constant.py is used to calculate the fringe constant for the unnotched beam

fringe_stresses is used to calculate the stresses at each fringe for the unnotched beams, given the fringe constant from fringe_constant.py

Vnotch_stress is used to calculate the analytical max stress for the V-notch

Unotch_stress is used to calculate the analytical max stress for the U-notch
